kuaishou.py
  
  快手视频批量下载，快手安卓版抓包，然后基于“发现”栏目，多线程下载。

------------------------------------------------------------------------------------------------------------------------------------------

  音悦台mv下载，图形界面，支持选择分辨率，默认下载到当前目录。

gui_qiubai.py
